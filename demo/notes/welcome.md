# Welcome to NextNote

This is a plain markdown file. NextNote stores every note exactly like this — no database lock-in, no proprietary format. Open it in any text editor, version-control it with git, sync it with iCloud / Dropbox / syncthing, whatever.

## What's in this sidebar

- **Notes** (top) — the folder you're reading right now. `.md` files, plus any subfolders you create. Drag, rename, delete via right-click.
- **Ebooks** (bottom tray) — `.epub` files. Click a book to open in the reader. Sidebar shows its table of contents.
- **Media** (bottom tray) — audio + video. Click to play in the ambient bar.

## Try these

1. **Edit this file.** Type anywhere. Auto-saves every 30 seconds.
2. **Split preview** — toolbar above the editor → click the split icon. Your markdown renders live on the right. Math works too: $e^{i\pi} + 1 = 0$.
3. **Focus mode** — **⌘⇧\\** hides all chrome. Escape exits.
4. **Open the Atomic Habits⸺ no wait, Alice** — click it in the Ebooks tray. Edge-click, arrow keys, or space paginate.
5. **Play the demo clip** — click sample-clip under Videos. Plays in the ambient bar at the bottom. **⌘⇧M** for the full player.
6. **Ask AI** — **⌘⇧I** opens the AI panel. See [ai-setup.md](ai-setup.md) to configure a provider.
